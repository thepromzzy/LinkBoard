# **App Name**: LinkBoard

## Core Features:

- Add Links: Users can add links with custom labels and icons to the dashboard.
- Custom Icons: Allow users to upload custom icons for each link.
- Drag and Drop: Enable users to reorder the links via drag and drop.
- Grid Layout: Display links in a responsive grid layout, adapting to different screen sizes.
- Local Storage: Store the dashboard configuration (links, icons, order) in the browser's localStorage.
- Responsive Design: Ensure the dashboard is fully responsive and works well on both desktop and mobile devices.

## Style Guidelines:

- Primary color: Light sky blue (#87CEEB) to evoke a sense of calm and accessibility.
- Background color: Very light blue (#E6F7FF), creating a subtle, soft backdrop that enhances the glassmorphism effect.
- Accent color: Soft lavender (#E6E6FA) for interactive elements like buttons and hover states.
- Body and headline font: 'PT Sans' sans-serif font to maintain a balance of modern appeal and readability.
- Use simple, line-based icons that are easily recognizable and consistent with the modern glassmorphism aesthetic.
- Implement a flexible grid layout that adjusts according to screen size, maintaining optimal spacing and alignment of elements.
- Subtle transitions and hover effects to provide visual feedback without overwhelming the user, such as a slight scale-up or color change on hover.
